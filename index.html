⚠️
Known data limitation: The Pipedrive API does not return archived leads. Export archived leads from Pipedrive’s reporting tool and use “Update Archived Leads” to sync them to Airtable. The dashboard combines these with live API data on every load.
📊
Deals
• Marketplace origin → Website / Call Booking
• ManuallyCreated origin + no Lead Created Date → Manually Added
• ManuallyCreated origin + Lead Created Date set → Converted from Lead
• Chatbot origin → Chatbot
• WebForms origin → Conference (channel ID = form name)
• API origin → API / Integration

Leads (API)
• channel_id = Zapier → Website / Call Booking
• channel_id contains chatbot → Chatbot
• channel_id contains surfe → Surfe / LinkedIn ⚠️
• channel_id = a web form name → Conference (form name shown)
• no channel_id → Manually Added

Archived Leads (Airtable)
• Source Channel = Zapier → Website / Call Booking
• Source Channel contains chatbot → Chatbot
• Source Channel contains surfe → Surfe / LinkedIn ⚠️
• Other Source Channel value → Conference (value shown)
• No Source Channel → Manually Added

Sync Archived Leads to Airtable

Export archived leads from Pipedrive: Reports → Leads → filter by Status = Archived → Export CSV.
This will overwrite all previous data in Airtable.
