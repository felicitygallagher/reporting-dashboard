LIVE DATA

⚠️
Known data limitation: The Pipedrive API does not return archived leads. To include them, export archived leads from Pipedrive’s reporting tool and use the “Update Archived Leads” button above to sync them to Airtable. The dashboard combines these with live API data on every load. Archived leads that have never been synced will not appear in the counts.
📊

Sync Archived Leads to Airtable

Export archived leads from Pipedrive: Reports → Leads → filter by Status = Archived → Export CSV.
This will overwrite all previous data in Airtable and replace it with the new export.
